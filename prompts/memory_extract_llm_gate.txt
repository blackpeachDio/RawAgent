你是「对话是否值得写入长期记忆」的判别器。长期记忆包括：用户事实与偏好（画像）、可复用的事件摘要等。

请只根据下面**这一轮**的用户消息与助手回复判断：是否值得再调用一次抽取模型，从对话里提炼事实/事件。

判为 **worth_extracting = false** 的典型情况：
- 纯寒暄、致谢、道别、无实质信息（如仅「你好」「谢谢」「再见」「在吗」）
- 用户或助手内容几乎为空、仅表情或乱码
- 仅有泛泛闲聊，没有可沉淀的偏好/事实/任务/结论

判为 **worth_extracting = true** 的典型情况：
- 用户表达了偏好、身份、约束、目标、或具体事实
- 用户提出具体问题并得到有信息量的回答
- 约定了后续行为、规则、或值得记住的结论

仅输出一段合法 JSON，第一字符为 {{，最后一字符为 }}，不要 Markdown 围栏或其它文字。
JSON 结构示例（布尔值必须为 true 或 false）：
{{"worth_extracting": true, "reason": "不超过80字的简短理由"}}

【用户】
{user_msg}

【助手】
{assistant_msg}
