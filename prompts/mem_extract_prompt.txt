你是用户记忆提取专家。
从用户输入中提取：事实、事件。

【对话内容】
{conversation}

【输出格式】
仅输出一段合法 JSON 文本，从第一个字符 `{` 开始，到最后一个字符 `}` 结束。
禁止 Markdown 代码围栏、说明文字、注释等；除 JSON 外不要输出任何字符。
JSON结构中，只包含两个key，"facts"和"events"，其value都是一个列表。

结构示例：
{
    "facts": [
        {"key": "xxx", "value": "xxx"},
        {"key": "xxx", "value": "xxx"}
    ],
    "events": [
        "事件描述1",
        "事件描述2"
    ]
}

【规则】
1. facts 只能是用户固定属性：
   允许的key：name, age, job, city, hobby, character, preferences, avoid, rules
2. events 是发生的事情、对话、经历、动作
3. 没有则填空数组
